### 缩略语表

| 缩略语     | 英文全称                                           | 中文全称              |
| ------- | ---------------------------------------------- | ----------------- |
| RuBP    | ribulose-1,5-biphosphate                       | 核酮糖-1,5-二磷酸       |
| Rubisco | ribulose-1,5-biphosphate carboxylase/oxygenase | 核酮糖-1,5-二磷酸羧化/加氧酶 |
| 3-PGA   | 3-phosphoglycerate                             | 甘油酸-3-磷酸          |
| GAP     | glyceraldehyde-3-phosphate                     | 甘油醛-3-磷酸          |
| DHAP    | dihydroxyacetone phosphate                     | 磷酸二羟丙酮            |
### RuBP 的羧化
- 羧化分为两步进行：水解+羧化
- 羧化由 **Rubisco** 催化进行
![[RuBP的羧化]]

### C<sub>3</sub> 产物的还原
- 3-PGA 先被 **ATP** 磷酸化"激活"，后在 NADPH 的作用下被还原为 **GAP**
- 光反应中产生的 **ATP、NADPH** 中的能量被转移至 GAP 中
- GAP 可异构化为 DHAP，统称为**磷酸丙糖**
![[三碳产物的还原]]
### RuBP 的再生
- 反应生成的 GAP 在一系列反应下，消耗 ATP，生成 RuBP
