### before you read files in...
读入文件前，需要明确：
1. 数据是否结构化
2. 文件的绝对路径 `盘符:/path/to/file`
3. 文件的分隔符类型
4. 文件是否表头（header），即行名（第一列）和列名（第一行）

### 使用命令读取
> [!important]
> 使用 `read` 系列函数可以读取文件

```r
read.table(file = "filename", sep = "\t", header = TRUE, row.names = 1)
```

或者直接使用：
```r
read.csv("filename", row.names = 1)
```

### 使用鼠标读取
在右上角视窗中 `environment` 选项内，点击 `import dataset` 可以直接读入文件

> [!summary]
> 使用 `read.csv` 读取文件最简单


### excel 文件的读写
excel 存在“工作簿”和“工作表”的概念：
- 对于只有一个工作表的 excel 文件，另存为 `csv` 进行处理就相当方便
- 对于一般的 excel 文件，可以直接从 import dataset 中进行读入，也可以使用 `openxslx` 包进行简单的读写操作

#### openxlsx 的使用
使用 openxlsx 读入文件，指定文件名和表格号
```r
read.xlsx("filename", sheet = 1)
```

使用 openxlsx 同理

```r
write.xlsx(iris, "iris.xlsx")
```
