### 通用转录因子的分类
- **与 TATA 等结合**
	- TFIID
		- TBP  (*TATA Box Binding Protein*) 
		- TAF（*TBP-associated factor*）
- **与 BRE 结合**
	- TFIIB[^1] 
- **与 RNAP**结合
	- TFIIF
- 其他
	- TFIIA
	- TFIIE
	- TFIIH

### 通用转录因子的功能
- 很多通用转录因子的功能可以通过名字识别 (TBP, TAF...) 
- 建议在 [[Preinitiation Complex|PIC形成的过程]] 中理解通用转录因子的功能

| 转录因子            | 功能                                     |
| --------------- | -------------------------------------- |
| TFIID, TBP, TAF | 识别 TATA 与下游启动子元件                       |
| TFIIB           | 识别 BRE，控制转录方向                          |
| TFIIE           | 结合并调控 TFIIF                            |
| TFIIF           | 稳定 RNAP 与 TFIIB，TBP 的互作，募集 TFIIE TFIIH |
| TFIIH           | 引发启动子溶解，磷酸化 CTD 并促进启动子逃逸               |
[^1]: BRE 就是 TFII*B* 识别元件
