Linux 里的文件拓展名是没有特殊含义的

以 `.gz` 文件为例（见[[压缩#`gzip` 命令]]），将文件命名为 xx. gz 虽然将让计算机标红这个文件

但其本质仍取决于用户对它的操作，不是说命名为. gz 就成了压缩文件了

真 `.gz` 文件，使用 `zcat` 输出如下
```bash
zhoux@westnorth:~/linux_practice/compress$ zcat tbc.1.txt.gz |head -5
i love my cat
/home/zhoux/
/home/zhoux/linux_practice
/home/zhoux/linux_practice/xibei.dir
/home/zhoux/linux_practice/xibei.dir/log
```

直接命名 `.gz` 文件，使用 `zcat` 只能报错
```bash
zhoux@westnorth:~/linux_practice/compress$ zcat tbc.gz
gzip: tbc.gz: not in gzip format
```
