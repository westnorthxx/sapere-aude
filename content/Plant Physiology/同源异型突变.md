花的某一重要器官位置由另一类器官替代的突变，如花瓣部位被雄蕊替代等，这种遗传变异现象称花发育的**同源异型突变**（homeotic mutation），即*本来应该是 A, 结果发育成了 B 的现象*

### A 缺失
A 缺失状态下的表达模式与突变状态为：

| 位置（轮数） | 表达情况  | 发育状态 |
| ------ | ----- | ---- |
| 1      | C     | 心皮   |
| 2      | B + C | 雄蕊   |
| 3      | B + C | 雄蕊   |
| 4      | C     | 心皮   |


### B 缺失
B 缺失状态下的表达模式与突变状态为：

| 位置（轮数） | 表达情况 | 发育状态 |
| ------ | ---- | ---- |
| 1      | A    | 萼片   |
| 2      | A    | 萼片   |
| 3      | C    | 心皮   |
| 4      | C    | 心皮   |
