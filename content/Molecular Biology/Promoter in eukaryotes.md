### 核心启动子的定义
- Pol III 转录机器能精确识别的最少的一组元件
### 核心启动子元件的分类——上游到下游
- TSS 上游
	- **BRE**：TFIIB 识别原件
	- **TATA** Box
- TSS
	- Inr：起始子
- TSS 下游
	- 下游启动子元件
		- DCE
		- DPE
		- MTE
### 核心启动子的结构
- 核心启动子不一定含有以上全部原件
- Inr 原件为最常见的原件
- TATA 与 DCE 一般一起出现
- TATA 与 DPE 一般不一起出现
### 核心启动子与通用转录因子
[[General transcription factors]]

| 启动子元件 | 转录因子  |
| :---- | ----- |
| BRE   | TFIIB |
| TATA  | TBP   |
| Inr   | TFIID |
| DCE   | TFIID |
| DPE   | TFIID |
