### `mv` 命令
使用 `mv` 命令可对文件进行移动，也就是**剪切**

用法与[[文件的复制]]很相似

其实，`mv` 命令等价于 `cp` + `rm 源文件`

> [!TIPS]
> 使用 `mv` 命令可以对文件进行重命名

```bash
zhoux@westnorth:~/linux_practice/files$ mv xb.R1.fastq ./xb.is.my.cat
zhoux@westnorth:~/linux_practice/files$ ls
2021.matrix  2022.matrix  copy  scRNA.script.R  script.R  script.r  xb.R2.fastq  xb.is.my.cat  xb.scRNA.fastq
#将xb.R1.fastq 重命名为 xb.is.my.cat
```
