### 基因概览
在典型的茎尖分化调控模型中，有如下几个重要的基因发挥工作

| 基因名    | 全称  | 类型          | 定位   | 功能                    |
| ------ | --- | ----------- | ---- | --------------------- |
| *WUS*  |     | homeobox TF | 组织中心 | 定位中心区，诱导干细胞产生         |
| *CLV1* |     | 受体激酶        | 组织中心 | 接受来自*CLV3* 的信号        |
| *CLV2* |     | 类受体蛋白       | CZ   | 与*CLV1* 形成 dimer 发挥作用 |
| *CLV3* |     | 多肽配体        | CZ   | 信号分子，负调控干细胞数量         |
| *STM*  |     | KNOX TF     | 全SAM | 维持分生组织的未分化状态          |
|        |     |             |      |                       |
### 机制 1：*WUS-CLV3* 负反馈
*WUS-CLV3* 的负反馈调节是维持 SAM 中干细胞数的关键：
1. *WUS* 表达，并在**其上层**的组织中确定**中心区**细胞，这种“告诉别的细胞：你是干细胞”的特性维持功能与[[静止中心]]细胞类似
2. *WUS* 的靶细胞（干细胞）内，*CLV3* 被诱导表达。*CLV3* 作为干细胞一种信号分子，反应了**干细胞的数量**
3. *CLV3* **向下层**扩散，与受体*CLV1* 结合，引发磷酸化级联反应，抑制*WUS* 表达
4. *WUS* 减少，干细胞数也因此稳定

$$
\text{WUS}\xrightarrow{促进}\text{CLV3}\xrightarrow{移动, 结合}\text{CLV1}\xrightarrow{抑制}\text{WUS} 
$$

以上负反馈位置了**干细胞数量稳定**

### 机制 2：*STM* 维持 SAM 状态
*STM* 基因在全局上维持 SAM 的未分化，快速分裂的状态
1. *STM* 抑制分化基因的表达，保证 SAM 不分化未其他器官
2. *STM* 上调 **CK** 表达，下调 GA 表达，使得 SAM 中细胞分裂而不分化、伸长

$$\text{STM} \xrightarrow{\text{抑制}} \text{分化基因 (AS1等)}$$


> [!summary]
> - **STM** 划定了一个“非叶片区域”，确立了分生组织的**身份**。
> - **WUS** 在这个区域的中心赋予上层细胞**干细胞特性**。
> - **CLV3/CLV1** 充当**计数器**，通过负反馈回路限制 WUS 的活性，维持干细胞数稳定。
