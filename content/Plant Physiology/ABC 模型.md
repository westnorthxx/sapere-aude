### 基因加法
对于一个正常双子叶花，A, B, C 基因组合对[[花的结构]]的影响如下表

| 轮数  | 结构  | 基因组合  |
| --- | --- | ----- |
| 1   | 萼片  | A     |
| 2   | 花瓣  | A + B |
| 3   | 雄蕊  | B + C |
| 4   | 心皮  | C     |
**当且仅当**基因按如下模式进行表达时，花的各轮器官才能正常表达

### 空间表达模式与互斥关系
对于 A, B, C 三类基因，其表达具有空间特异性

| 基因  | 表达范围（轮数） |
| --- | -------- |
| A   | 1 - 4    |
| B   | 2, 3     |
| C   | 1 - 4    |
然而，这与表观上的表达模式不符，这源于 A, C 间的互斥关系：
- 在 1，4 轮，A，C 分别占据优势，各自表达
- 在 2 轮，A 基因抑制 C 基因，仅 A 表达
- 在 3 轮，C 基因抑制 A 基因，仅 C 表达
