### 保守的识别结构域
细胞内信号蛋白的相互作用靠蛋白质**模式结合域**特异性介导，它保证了在细胞**拥挤的热力学环境下**，信号的识别依旧高效有序

模式结构域不具酶活性，但能特异性识别特定基序或蛋白质上特定的修饰位点

模式结构域与对象的**亲和性较弱**，这有利于快速和反复进行精细的组合式网络调控，也符合信号控制的**可逆性原则**

常见的模式结构域有：

|  结构域  |   例子    |
| :---: | :-----: |
|  SH2  |   Src   |
|  SH3  |   Src   |
|  PTB  |   Shc   |
|  PDZ  |   FAP   |
|  WW   |  Nedd4  |
|  PH   | Akt，SOS |
| FYVE  |  EEA1   |
| 死亡结构域 |   Fas   |
