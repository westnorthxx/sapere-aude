### 基本语法
> [!important]
> **主命令**（空格）*选项*  参数
#### 主命令
标识执行的功能
#### 选项
标识执行方式
#### 参数
标识执行对象/细节

```bash
ls -l /home/zhoux
# ls 是主命令，即list
# -l 是选项
# /home/zhoux 是参数
```

### 基本命令（感觉没什么用的）
#### who 命令
```bash
who
# 返回服务器上的用户
who am i
# 返回你的用户信息
whoami
# 返回你的名字
w
# 返回详细的用户信息
```
#### uname 命令
```bash
uname -i -n
uname -in
# 返回用户名和CPU架构, 以上两命令等效
```

> *命令的特点*
> - 一个命令可以有**多个选项**
> - 一个命令的选项输出有优先级

#### clear 命令
```bash
clear
# 清除屏幕，即拉动进度条值没有东西的地方
```
#### 用户命令
```bash
su xibei
# 切换用户
sudo useradd xibei
# 添加用户
sudo passwd xibei
# 设置密码
```
