> [!important]
> `less` 命令对于长文本相当便利


### `more` 命令
`more` 命令可以进入一种文本浏览模式，类似于在一次输出的内容里加入了可以控制的 scroll

```bash
more history.txt
```
在 `more` 命令模式下，**方向键**可以控制上下，有如下操作

| 操作   | 效果      |
| ---- | ------- |
| `:f` | 线上当前文件名 |
| `q:` | 退出      |
| `h`  | 帮助      |

### `less` 命令
`less` 相比 `more` 更符合一般文档的使用习惯，基本和 Windows 文本文件没有差别

有如下重要操作

| 操作        | 效果        |
| --------- | --------- |
| `G` , `g` | 跳到文末，跳到开头 |
| `/字符`     | 搜索字符      |
| `n` / `M` | 向下/向上匹配结果 |
