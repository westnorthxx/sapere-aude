### 创建向量
使用 `c()` 函数创建向量，使用逗号分隔
```r
a <- c(1，2，3，4，5)
b <- c("A" , "B" , "C")
c <- c(TRUE , FALSE)
```


一些其他语法和函数，如 `seq()` 函数和 `rep()` 函数也可以快速创建向量
```r
b <- 5:1
c <- seq(1, 100, by 2)
d <- seq(100, 1, by -2)
d <- rep(c(1,2,3) , times = 10) #1,2,3重复10次
e <- rep(c(1,2,3) , each = 10) #1,2,3分别重复10次
```


> [!important]
> 凡是对于单个值的选项，使用 `c()` 都有可能可以替换称一组值

例如，想要生成 A1-A10，B1-B10 模式的向量，可以如下实现
```r
a <- rep(c('A' , 'B') , each = 10)
b <- seq(1 , 10)
c <- paste(a , b , sep = "")
```

> 字符串向量中注意加上引号，单、双引号都行
