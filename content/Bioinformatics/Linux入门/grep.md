**`grep`** 为 Global Regular Expression Print，也可以理解为*grab*，即从每行中抓取特定内容

**`grep`** 可以单纯进行字符对字符的匹配，更可以搭配[[正则表达式]]进行更大范围的匹配

### 选项
`grep` 有如下重要选项

| 选项   | 全称      | 含义          |
| ---- | ------- | ----------- |
| `-c` | count   | 计量匹配行数      |
| `-i` | ignore  | 忽略大小写       |
| `-n` | number  | 标出匹配行号      |
| `-v` | verbose | 列出未匹配项目     |
| `-w` | whole   | 要求完全匹配      |
| `-l` |         | 列出带有匹配行的文件名 |
