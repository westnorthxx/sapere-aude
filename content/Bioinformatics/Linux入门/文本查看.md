创建一个 1-100，每行一个数字的文本文件
### `wc` 命令
`wc` 意味 word count，用于计量行数，单词数，字符数
```bash
wc test.txt
100 100 292 test.txt#注意输出结果的顺序
```
### `cat` 命令
`cat` 为 concatenate，已经很熟悉了，注意一些选项

| 选项   | 含义                  |
| ---- | ------------------- |
| `-A` | 显示所有字符（包括特殊字符，如换行符） |
| `-s` | 合并空行：多个空行合为一行       |
| `-b` | 显示行号                |

通过重定向，`cat` 也可以进行写入
- `cat` 有内容的文件，则是简单的输出重定向
- `cat` 新建文件，则可以自行写入
	- `CTRL D` 可以退出

### `head` 和 `tail`
分别用来输出文件的前/后几行，-n 可以选择输出行数

注意与[[管道符]]的联用，可以输出中间几行
