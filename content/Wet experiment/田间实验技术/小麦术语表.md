### 简介
掌握常见的术语可以显著提高阅读文献的效率并改善阅读文献的体验。本文列举小麦研究领域常用的**形态学术语**和**发育时期术语**，并提供对应中文翻译和解释。

本文翻译自*WHEAT TRAINING* 网站提供的[术语表](https://www.wheat-training.com/wp-content/uploads/Wheat_growth/pdfs/Glossary.pdf)，并可能在日后不断增加。

### 形态学术语 Morphological terms

|      术语(英)      | 术语(中) |                    释义                    |
| :-------------: | :---: | :--------------------------------------: |
|   **Anther**    |  花药   |              花朵的雄性（产生花粉的）部分              |
|     **Awn**     |   芒   |              麦穗上的刺，从小花处延伸出来              |
| **Coleoptile**  |  胚芽鞘  |              覆盖并保护幼苗尖端的保护鞘               |
|  **Flag leaf**  |  旗叶   |           抽穗前的最后一片叶子。位于穗的正下方。            |
|   **Floret**    |  小花   | 每一朵独立的花（包含花药和柱头）。几个小花（通常为 3 个）组成一个**小穗** |
|    **Ovary**    |  子房   |         未成熟、未授粉的籽粒前体。由胚与其周围的胚乳组成         |
|  **Penduncle**  |  穗柄   |     支撑穗的茎。穗柄只有一小部分暴露在外，大部分隐藏在多层叶鞘之下      |
|   **Radicle**   |  胚根   |    发育中幼苗从种皮中长出的第一部分，随后发育为根部组织。也被称为鸡爪根    |
|    **Spike**    |   穗   |     麦穗。一个穗通常由 20 个左右的**小穗**着生在穗轴上组成      |
|  **Spikelet**   |  小穗   |         小麦花的基本单位，一般含有 3 朵**小花**          |
|   **Stigma**    |  柱头   |              花朵的雌性（接受花粉的）部分              |
|   **Tiller**    |  分蘖   |          从小麦基部分生组织发出的芽，是一种特殊分支           |
|   **Glumes**    |  颖片   |        位于小麦小穗最外侧的一对苞片，包裹并保护内部的小花         |
|  **Internode**  |  节间   |       小麦茎秆上相邻两个节之间的部分，其伸长直接影响植株高度        |
| **Leaf sheath** |  叶鞘   |       叶片基部包裹在小麦茎秆上的鞘状部分，具有支撑和保护作用        |
| **Leaf blade**  |  叶片   |        叶片向外伸展的平扁部分，是小麦进行光合作用的主要器官        |


### 生育时期术语 Developmental timing terms

|       术语(英)       | 术语(中) |                          释义                           |
| :---------------: | :---: | :---------------------------------------------------: |
|   **Anthesis**    |  开花期  |      花药成熟并进行授粉的时间。开花首先从**穗**中心的小花开始，然后想穗的基部的顶端外移      |
| **Emasculation**  |  去雄   |                从花中除去**花药**以防止自花授粉的的操作                 |
|    **Heading**    |  抽穗   | 麦**穗**从**旗叶鞘**中伸出的生长阶段。国内定义为田间 50% 的个体穗伸出 1cm（不含芒）的日期 |
|  **Imbibition**   |  吸涨   |             种子在发芽前吸水的行为。种子在吸涨后重新获得完整的代谢活性             |
|  **Pollination**  |  授粉   |                 花粉沉积在柱头上并导致受精与新种子的产生                  |
|  **Senescence**   |  衰老   |         植物的老化过程，开始于新籽粒开始形成时；植物的颜色将从绿色变为金色/黄色          |
| **Vernalisation** |  春化   |        将小麦暴露在低温以诱导开花。这模拟了小麦开花需要的环境信号，对冬小麦品种很重要        |
